# 狼人杀基础知识

## 核心术语

- **金水**: 预言家查验确认的好人。可信度高。
- **查杀**: 预言家查验确认的狼人。应当推动投票淘汰。
- **银水**: 女巫夜间救的人。有一定可信度但不绝对。
- **悍跳**: 狼人假冒预言家，跳出来争夺话语权。
- **对跳**: 两人都声称自己是预言家，一真一假。村民必须选边站。
- **警徽流**: 预言家公布的查验计划——今晚验谁，如果我死了，村民按这个顺序查。
- **站边**: 在对跳中选择相信某一方。
- **冲票**: 狼人协调集体投票某人。
- **自刀**: 狼人夜间杀自己的队友，制造混乱或骗取女巫解药。
- **倒钩**: 狼人假装站边真预言家，暗中破坏。
- **深水狼**: 低调潜水的狼人，不主动说话，不引人注意。
- **排狼坑**: 逐步排除，缩小狼人嫌疑范围。
- **心路历程**: 预言家解释为什么查这个人、怎么判断的——越详细越可信。

## 阶段流程

1. **夜晚**: 狼人杀人 → 预言家查验 → 女巫救/毒 → 守卫守人
2. **白天**: 公布夜间结果 → 自由发言 → 投票淘汰
3. 循环直到一方获胜

## 胜负条件

- **好人胜**: 所有狼人被淘汰
- **狼人胜**: 狼人数量 ≥ 好人数量

## 核心博弈原则

- 信息就是权力。预言家的查验结果是好人阵营最重要的资源。
- 第一天的信息最少，博弈最激烈。预言家跳身份、狼人是否悍跳，决定了整局走向。
- 投票模式暴露阵营——分析谁和谁投票一致，可以推断同伙关系。
- 发言风格分析：过度解释=心虚，沉默不语=可疑，逻辑混乱=可能在编故事。
